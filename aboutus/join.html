招聘职位

媒体运营推广专员（1人）

岗位职责：

  1. 1、协助制定产品推广方案和策略，利用各种互联网资源、相关行业网络媒体、论坛、微信、微博等方式进行产品推广执行；
  2. 2、协助撰写公司产品宣传文案及新闻稿，负责微信，微博及各类对外宣传文案；
  3. 3、维护运营类产品的网页，以及公司网站、微信、微博等自媒体平台，建立有效运营手段提升平台知名度、用户活跃度、美誉度；
  4. 4、负责运营类网站的SEO推广，竞价排名，宣传等工作；
  5. 5、监控各平台社会化营销效果，分析并形成阶段统计报告，不断优化提升推广能力

岗位要求：

  1. 1、有1-3年的媒体、市场或产品的相关经验；
  2. 2、本科以上学历，市场营销、新闻媒体、中文类专业优先选择；
  3. 3、了解各种媒体发布形式，有一定媒体资源，具备优秀的沟通能力，能与多方媒体保持良好的合作关系；
  4. 4、熟悉SEO、SEM、微信、微博等自媒体运营、有策划、文案功底；
  5. 5、对市场工作充满热情，具有想象力和创造力，工作积极严谨，执行力强，有责任心。
工作地址：深圳        简历投递邮箱：hr@yg-ai.com

运营经理（2人）

岗位职责：

  1. 1、负责产品及品牌推介及实施，策划具有高话题性的网络品牌及营销推广方案，对执行效果负责；
  2. 2、建立和维系媒体关系，定期宣传产品和品牌正面形象；
  3. 3、能够根据不同的传播特点制定相应的媒体宣传策略，可独立撰写媒体传播方案及媒体新闻，有效运用内部、外部资源，使品牌影响范围最大化；
  4. 4、负责公关事件危机处理及公关舆情监控督导。

岗位要求：

  1. 1、本科及以上学历，广告策划、市场营销或新闻传播学等相关专业；
  2. 2、3年以上互联网营销推广工作经验；
  3. 3、熟悉微信、视频、微电影、微博、论坛、SNS网站等社会化媒体渠道，熟悉互联网营销手段，具有良好的人脉网络；
  4. 4、具有出色的活动策划、突出的媒介公关能力，善于话题营销、事件营销，并能提供成功案例；
  5. 5、不拘泥于固有形式、有创新欲望，对互联网流行元素有较强的接受学习能力；
  6. 6、具备较高的职业素养、敬业精神及团队精神，擅于沟通协作。
工作地址：深圳        简历投递邮箱：hr@turingcat.com
